<h1>FIM File Integrity Monitor</h1>


<h2>Description</h2>
In this lab i setup a File Integrity Monitor to inspect and monitor files continuously  to make sure if any changes ( authorized or unauthorized) has been made to the file. I use a PowerShell script to monitor folders alert and log any changes made in the folder.
<br />


<h2>exposure to:</h2>

- <b>PowerShell</b> 
- <b>Hashing Algorithms (SHA-512)</b>
- <b>Automation </b>

<h2>Environments Used </h2>

- <b>Oracle VirtualBox</b>
- <b>Windows 10</b>
